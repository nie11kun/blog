Linux 设置开机自启动 python/bash 脚本

  • 作者: admin
  • 时间: 2019-10-15
  • 分类: Linux
  • 评论

打开 rc.local 文件：

vim  /etc/init.d/rc.local

在 rc.local 后加入启动命令：

sudo python /your/path/script.py &

&符号为让脚本后台运行，不在命令行显示结果.

typecho 修改默认上传文件大小限制

  • 作者: admin
  • 时间: 2019-08-15
  • 分类: Linux,web,software
  • 评论

修改 php.ini 文件，我使用的是 php5.6 版，所以首先查找 fpm 的配置文件：

locate php.ini

找到后打开文件，如：

vim /etc/php5.6/fpm/php.ini

使用 / 搜索字段，修改如下：

upload_max_filesize = 1024M;
post_max_size = 1024M;
max_execution_time=600;

重启 php 服务：

service php5.6-fpm restart

nginx 配置中 http 段增加如下：

client_max_body_size 50m;

重启 nginx：

service nginx reload

chown 和 chmod 管理文件权限

  • 作者: admin
  • 时间: 2019-04-19
  • 分类: Linux
  • 评论

Linux 的文件属性主要包含两部分：ownership 和 permission

一个文件或文件夹的属性包括：1.归属于某个 user 和 user group，2.在此 user 和 user group 下的读/写/执行权限及其他 user 的相关权限。分别通过 chown 和 chmod 来进行设置。
