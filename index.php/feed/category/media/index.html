快捷键功能
H隐藏调出的线条锚点等信息
O显示笔刷画过的区域
J显示过爆或过黑区域
/切换显示原始图像和当前图像
Kadjustment 笔刷工具
Glibrary 窗口下切换到 grid 网格图片模式
P标记照片为 picked
U清除照片的 picked 或 rejected 标记
X标记照片为 rejected
alt 点击 white / black快速找到黑场/白场
alt 拖动锐化/降噪滑块去掉饱和度只显示黑白，调整起来更清晰
alt 笔刷减去画出的地方
shift 笔刷点击起点终点画一条直线
shift +隐藏调整窗口 只显示图片
shift -退出只显示图片模式
